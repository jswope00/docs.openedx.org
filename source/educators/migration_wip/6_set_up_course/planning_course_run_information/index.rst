#####################################
6.2 Planning Course Run Information
#####################################

Course run information can change each time a course is run. This section lists
the optional and required information for course runs, as well as guidelines to
help you create an effective course and About page.

After you have determined the information for your course run, you enter this
information in Studio. For more information, see :ref:`Creating a Course <Creating a New Course>`.
